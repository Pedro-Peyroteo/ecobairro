**Decisão — `SENSOR_RECUPERADO`:** Quando um sensor que estava `OFFLINE` volta a enviar leituras válidas, cria-se um alerta de tipo `SENSOR_RECUPERADO` com severidade `INFO`. Isto fecha o ciclo de vida do alerta e permite calcular o TMEF (Tempo Médio Entre Falhas) por dispositivo.
