**Fluxo de atribuição automática:**

```
Evento disparador (ex: quiz concluído com score 100)
  │
  └── BullMQ job: 'badge.evaluate'
        payload: { cidadao_id, evento_tipo: 'QUIZ_SCORE_PERFEITO', contexto: {...} }
        │
        ├── SELECT badges WHERE criterio_tipo = 'QUIZ_SCORE_PERFEITO' AND ativo=true
        ├── Para cada badge elegível:
        │     SELECT 1 FROM cidadao_badges WHERE cidadao_id=? AND badge_id=? (já ganhou?)
        │     SE NÃO: verificar critério completo contra dados do cidadão
        │     SE ELEGÍVEL: INSERT cidadao_badges
        │                  UPDATE cidadao_gamificacao SET score += pontos_recompensa
        │                  NOTIFY gamificacao_badge_earned, '{"cidadao_id":...}'
        │
        └── NestJS WS gateway emite notificação para o cidadão em tempo real
```
