**Tabela de histórico: `zonas_historico`**

Quando uma zona é alterada (RF-06 exige manutenção de histórico), regista-se o estado anterior:

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  tabela: zonas_historico                                                    │
├──────────────────────────┬──────────────────────┬───────────────────────────┤
│ id                       │ UUID                 │ PK                        │
│ zona_id                  │ UUID                 │ NOT NULL, FK → zonas(id)  │
│ snapshot                 │ JSONB                │ NOT NULL                  │
│                          │                      │ estado completo da zona   │
│                          │                      │ antes da alteração        │
│ alterado_por             │ UUID                 │ FK → users(id)            │
│ alterado_em              │ TIMESTAMPTZ          │ NOT NULL, DEFAULT now()   │
│ motivo                   │ TEXT                 │ nullable                  │
└──────────────────────────┴──────────────────────┴───────────────────────────┘

Índices: (zona_id, alterado_em DESC)
```

O `snapshot` em JSONB guarda o estado completo da linha antes de cada UPDATE — incluindo a geometria serializada. Isto evita uma segunda tabela de geometria histórica e simplifica a auditoria.

**Índices:**

```
┌──────────────────────┬──────────────────────────────────┬────────────────────┐
│ Coluna(s)            │ Tipo                             │ Uso                │
├──────────────────────┼──────────────────────────────────┼────────────────────┤
│ (geometria)          │ GIST                             │ ST_Within,         │
│                      │                                  │ ST_Contains,       │
│                      │                                  │ ST_Intersects      │
│ (ativa)              │ PARTIAL BTREE WHERE ativa=true   │ todas as queries   │
│                      │                                  │ operacionais       │
│ (tipo)               │ BTREE                            │ filtrar zonas IoT  │
│ (zona_id, alterado_em│ BTREE em zonas_historico         │ auditoria RF-06    │
│  DESC)               │                                  │                    │
└──────────────────────┴──────────────────────────────────┴────────────────────┘
```
