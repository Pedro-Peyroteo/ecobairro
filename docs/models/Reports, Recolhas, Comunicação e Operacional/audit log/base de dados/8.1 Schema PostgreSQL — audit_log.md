  (actor_id, criado_em DESC)         BTREE — acções de um utilizador

O `audit_log` é **append-only** — sem UPDATE nem DELETE. Registado automaticamente por middleware NestJS para todas as operações sensíveis: alterações de estado de reports, mudanças de role, acesso a dados sensíveis PII, gestão de dispositivos, e todas as acções de admin.
