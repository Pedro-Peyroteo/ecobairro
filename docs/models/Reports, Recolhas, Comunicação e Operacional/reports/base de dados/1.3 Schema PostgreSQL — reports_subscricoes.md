Cidadãos que subscrevem um report existente (RF-12 — "juntar-se" em vez de duplicar).

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  tabela: reports_subscricoes                                                │
├──────────────────────────┬──────────────────────┬───────────────────────────┤
│ coluna                   │ tipo                 │ constraints / notas       │
├──────────────────────────┼──────────────────────┼───────────────────────────┤
│ report_id                │ UUID                 │ NOT NULL                  │
│                          │                      │ FK → reports(id)          │
│ cidadao_id               │ UUID                 │ NOT NULL                  │
│                          │                      │ FK → users(id)            │
│ criado_em                │ TIMESTAMPTZ          │ NOT NULL, DEFAULT now()   │
└──────────────────────────┴──────────────────────┴───────────────────────────┘
```
