| #   | Método  | Rota                           | Descrição                                      | Auth                     | Fluxo                                                                |
| --- | ------- | ------------------------------ | ---------------------------------------------- | ------------------------ | -------------------------------------------------------------------- |
| CB1 | `GET`   | `/campanhas`                   | Listar campanhas activas na zona do cidadão    | CIDADAO                  | NestJS → PG réplica                                                  |
| CB2 | `GET`   | `/campanhas/:id`               | Detalhe com critérios e elegibilidade          | CIDADAO                  | NestJS → PG réplica                                                  |
| CB3 | `GET`   | `/campanhas/:id/elegibilidade` | Verificar se cidadão é elegível                | CIDADAO                  | NestJS → PG réplica (conta quizzes, recolhas — sem NIF/morada RF-21) |
| CB4 | `GET`   | `/admin/campanhas`             | Listar todas incluindo inactivas               | TECNICO_AUTARQUIA, ADMIN | NestJS → PG réplica                                                  |
| CB5 | `POST`  | `/admin/campanhas`             | Criar campanha com critérios observáveis RF-21 | TECNICO_AUTARQUIA, ADMIN | NestJS → PG write                                                    |
| CB6 | `PUT`   | `/admin/campanhas/:id`         | Editar campanha (só se não iniciada)           | TECNICO_AUTARQUIA, ADMIN | NestJS → PG write                                                    |
| CB7 | `PATCH` | `/admin/campanhas/:id/ativar`  | Activar campanha e notificar zona              | TECNICO_AUTARQUIA, ADMIN | NestJS → PG write ativa=true → BullMQ notif zona                     |
