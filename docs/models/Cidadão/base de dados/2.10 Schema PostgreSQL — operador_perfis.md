```
┌─────────────────────────────────────────────────────────────────────────────┐
│  tabela: operador_perfis                                                    │
│  Relação: 1:1 com users WHERE role IN                                       │
│           ('OPERADOR_VEOLIA','TECNICO_AUTARQUIA','TECNICO_CCDR','ADMIN')    │
│  Contém: entidade empregadora, zona(s) de responsabilidade, cargo          │
├──────────────────────────┬──────────────────────┬───────────────────────────┤
│ coluna                   │ tipo                 │ constraints / notas       │
├──────────────────────────┼──────────────────────┼───────────────────────────┤
│ id                       │ UUID                 │ PK, gen_random_uuid()     │
│ user_id                  │ UUID                 │ NOT NULL, UNIQUE          │
│                          │                      │ FK → users(id)            │
│                          │                      │ ON DELETE CASCADE         │
├──────────────────────────┼──────────────────────┼───────────────────────────┤
│ nome_completo            │ VARCHAR              │ NOT NULL                  │
│ entidade_empregadora     │ VARCHAR(100)         │ NOT NULL                  │
│                          │                      │ ex: 'VEOLIA',             │
│                          │                      │ 'CAMARA_MUNICIPAL_AVEIRO',│
│                          │                      │ 'CCDR_CENTRO'             │
│ cargo                    │ VARCHAR(100)         │ nullable                  │
│                          │                      │ ex: 'Técnico de Recolha', │
│                          │                      │ 'Coordenador de Zona'     │
│ zonas_responsabilidade   │ UUID[]               │ NOT NULL, DEFAULT '{}'    │
│                          │                      │ array de zonas.id         │
│                          │                      │ atribuídas ao operador    │
├──────────────────────────┼──────────────────────┼───────────────────────────┤
│ criado_em                │ TIMESTAMPTZ          │ NOT NULL, DEFAULT now()   │

```

**Índices:**

```
┌──────────────────────────────────────┬────────────────┬──────────────────────┐
│ Coluna(s)                            │ Tipo           │ Uso                  │
├──────────────────────────────────────┼────────────────┼──────────────────────┤
│ (user_id)                            │ UNIQUE BTREE   │ join rápido com users│
│ (entidade_empregadora)               │ BTREE          │ filtrar por entidade │
│ (zonas_responsabilidade)             │ GIN            │ queries "operadores  │
│                                      │                │ desta zona" via @>   │
│ (nome_completo)                      │ GIN pg_trgm    │ pesquisa por nome    │
└──────────────────────────────────────┴────────────────┴──────────────────────┘
```

**Notas de design:**
- `zonas_responsabilidade` é um array UUID[] com índice GIN. A query típica é: `SELECT * FROM operador_perfis WHERE zonas_responsabilidade @> ARRAY[:zona_id]::uuid[]` para encontrar operadores responsáveis por uma zona.
- Operadores são criados exclusivamente por ADMIN via `POST /admin/operadores` — não existe auto-registo de operadores.
- O soft delete do operador é controlado pela coluna `users.eliminado_em` — esta tabela não tem `eliminado_em` próprio.
