| #   | Método   | Rota                                      | Descrição                                       | Auth    | Fluxo                                          |
| --- | -------- | ----------------------------------------- | ----------------------------------------------- | ------- | ---------------------------------------------- |
| C1  | `GET`    | `/cidadaos/me/consentimentos`             | Listar consentimentos activos e histórico       | CIDADAO | NestJS → PG réplica (sem cache — audit trail)  |
| C2  | `POST`   | `/cidadaos/me/consentimentos`             | Dar consentimento para finalidade               | CIDADAO | NestJS → PG write com timestamp + versão       |
| C3  | `DELETE` | `/cidadaos/me/consentimentos/:finalidade` | Retirar consentimento (RNF-PRIV-03)             | CIDADAO | NestJS → PG write revogado_at → BullMQ cleanup |
| C4  | `POST`   | `/cidadaos/me/portabilidade`              | Exportar todos os dados pessoais (RGPD Art. 20) | CIDADAO | NestJS → BullMQ export job → email com link    |

**Corpo de C2:**

```

  finalidade: "controlo_quantidade_futura" | "marketing" | ...,

}
```
